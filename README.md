# Trader Behavior vs Market Sentiment

## Objective
Analyze how market sentiment affects trader performance and behavior.

## Methodology
- Data cleaning and preprocessing
- Feature creation (PnL, win rate, segments)
- Exploratory Data Analysis (EDA)
- Segmentation analysis

## Key Insights
- Fear increases risk-taking and volatility
- Frequent trading leads to inconsistent results
- High exposure leads to extreme outcomes

## Strategy Recommendations
- Reduce leverage during fear
- Control trade frequency
